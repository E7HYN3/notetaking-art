Если вы занимаетесь исследовательской работой, шаблоны помогут систематизировать данные и избежать хаоса.

Пример базового шаблона для исследования:

```
# Исследование: {{Название}}
## Вопросы  
- Основной вопрос:  
- Дополнительные вопросы:  
## Источники  
- Книги:  
- Статьи:  
- Сайты:  
## Результаты  
- Основные выводы:  
- Дальнейшие шаги:  
```

Этот шаблон помогает сосредоточиться на цели, хранить список источников и фиксировать выводы. Также можно добавлять ссылки на другие заметки для построения сети знаний.
